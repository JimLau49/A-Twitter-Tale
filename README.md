# A-Twitter-Tale

A Twitter-like website for our SOEN341 class

# Quick Start Guide
1. Clone the repo.
2. Navigate to the cloned A-Twitter-Tale directory.
3. Run ` python manage.py runserver `.
4. (optional) type http://127.0.0.1:8000/ in your browser.
